# frontend_v0

# Dev: Project set up

1. `npx create-next-app@latest frontend_v0`
   - TypeScript: Yes
   - ESLint: Yes
   - React Compiler: No
   - Tailwind: Yes
   - src directory: Yes
   - App Router: Yes
   - Import alias: No

2. cd frontend_v0

3. npm install @iconify/icons-ion @iconify/react aos next-themes nextjs-toploader nodemailer

4. npm install -D @tailwindcss/postcss @types/aos @types/nodemailer

5. npm run dev
